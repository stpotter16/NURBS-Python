Geometric Entities
^^^^^^^^^^^^^^^^^^

The geometric entities are used for advanced algorithms, such as tessellation. The :py:class:`.AbstractEntity` class
provides the abstract base for all geometric and topological entities.

This module provides the following geometric and topological entities:

* :py:class:`.Vertex`
* :py:class:`.Triangle`
* :py:class:`.Face`
* :py:class:`.Body`
